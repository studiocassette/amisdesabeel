Theme officiel pour le site des Amis de Sabeel France - Reseau de soutien au Centre oecumenique Sabeel.

== Description ==

Le theme Amis de Sabeel France est concu specifiquement pour l'association Amis de Sabeel France (ADSF),
un reseau de soutien au Centre oecumenique Sabeel, present a Jerusalem et Nazareth.

=== Fonctionnalites ===

* Design moderne et responsive
* Types de contenu personnalises (Documents, Vagues de prieres, Partenaires)
* Taxonomies personnalisees pour categoriser les documents
* Section Hero personnalisable
* Formulaire de newsletter integre
* Support du logo personnalise
* Menus personnalisables (principal, pied de page, legal)
* Zones de widgets
* Compatible avec l'editeur de blocs
* Optimise pour le referencement (SEO)
* Accessibilite amelioree

=== Types de contenu personnalises ===

1. **Documents** - Pour les traductions et publications
   - Categories: Sabeel, Kairos, Publications, Videos
   - Types: Analyse, Communique, Appel, etc.

2. **Vagues de prieres** - Pour les prieres hebdomadaires

3. **Partenaires** - Pour presenter les organisations partenaires
   - Types: International, France, Europe

== Installation ==

1. Telechargez le theme et decompressez-le
2. Uploadez le dossier `amis-sabeel-france` dans `/wp-content/themes/`
3. Activez le theme via le menu Apparence > Themes dans WordPress
4. Configurez le theme via Apparence > Personnaliser

== Configuration ==

=== Logo ===
Ajoutez votre logo via Apparence > Personnaliser > Identite du site

=== Menus ===
Creez vos menus via Apparence > Menus. Le theme supporte 4 emplacements:
- Menu principal (en-tete)
- Menu pied de page
- Menu A propos
- Menu Legal

=== Page d'accueil ===
Le theme affiche automatiquement la page d'accueil personnalisee avec:
- Section Hero
- Vague de priere de la semaine
- Actualites recentes
- Collections
- Newsletter

=== Personnalisation ===
Acces via Apparence > Personnaliser:
- Section Hero (titre, description)
- Pied de page (texte, mission)
- Contact (email, localisation)
- Reseaux sociaux
